区块元素

段落

一般使用一个空行，或者两个段尾空格来分段，也可以插入<br/>来分段。如果只是按Enter则无法分段。

标题

以1到6个#加上文字，就可以创建标题，对应1到6级标题。

# This is an H1

## This is an H2

###### This is an H6

引用

用>可以创建引用样式。比如：

西蒙·波伏娃，出生于巴黎一个天主教色彩很深的资产阶级家庭。她具有作家，哲学家、散文家、戏剧家等多重身份，是成为存在主义代表性的人物，女权运动的先驱，主要作品有《西蒙·波伏娃的美国纪行》、《满大人》、《老年》、《第二性》等。 > 这是嵌套引用

列表

## un-ordered list
*   Red
*   Green
*   Blue

## ordered list
1.  Red
2.  Green
3.  Blue

实现效果如下：

un-ordered list

  • Red
  • Green
  • Blue

ordered list

  1. Red
  2. Green
  3. Blue

任务

用 [ ] 或 [x] 可以任务清单。

- [ ] a task list item
- [ ] list syntax required
- [ ] normal **formatting**, @mentions, #1234 refs
- [ ] incomplete
- [x] completed

实现效果如下：

  • a task list item
  • list syntax required
  • normal formatting, @mentions, #1234 refs
  • incomplete
  • completed

代码块

用成对的三个`可以创建代码块，代码要放在中间。

def my_function(x, y, z=1.5):
    if z > 1:
        return z * (x + y)
    else:
        return z / (x + y)

数学公式

$$
\begin{alignedat}{2}
   10&x+ &3&y = 2 \\
   3&x+&13&y = 4
\end{alignedat}
$$

实现效果如下：

\[ \begin{alignedat}{2} 10&x+ &3&y = 2 \\ 3&x+&13&y = 4 \end{alignedat} \] 行内公式也支持 \(y=\textstyle\sum_{i=1}^n x\)。

表格

| Left-Aligned  | Center Aligned  | Right Aligned |
| :------------ |:---------------:| -----:|
| col 3 is      | some wordy text | $1600 |
| col 2 is      | centered        |   $12 |
| zebra stripes | are neat        |    $1 |

实现效果如下：

Left-Aligned Center Aligned Right Aligned
col 3 is some wordy text $1600
col 2 is centered $12
zebra stripes are neat $1
